## üìù Exercise 2.1: Finding Principal Values

The main goal of this exercise is to find the **principal value** of inverse trigonometric functions. This means finding the *unique angle* that lies within the defined **Principal Value Branch (Range)** for each function. We'll heavily rely on our memorized table of domains and ranges, and the negative argument properties.

---

### **Question 1: Find the principal value of $\sin^{-1} \left(-\frac{1}{2}\right)$**

**Thought Process:**
1.  **Identify the function and its principal range:** We have $\sin^{-1} x$. Its principal value branch (range) is $[-\frac{\pi}{2}, \frac{\pi}{2}]$.
2.  **Identify the value:** The input is $-\frac{1}{2}$.
3.  **Handle the negative:** Since it's negative, we'll use the property $\sin^{-1}(-x) = -\sin^{-1} x$.
4.  **Find the basic angle:** What angle has a sine of $\frac{1}{2}$? From our basic trig values table, $\sin\left(\frac{\pi}{6}\right) = \frac{1}{2}$.
5.  **Combine and check:** So, the angle is $-\frac{\pi}{6}$. Is $-\frac{\pi}{6}$ within $[-\frac{\pi}{2}, \frac{\pi}{2}]$? Yes, it is.

**Solution:**
Let $y = \sin^{-1} \left(-\frac{1}{2}\right)$.
Then $\sin y = -\frac{1}{2}$.

We know that the principal value branch of $\sin^{-1} x$ is $[-\frac{\pi}{2}, \frac{\pi}{2}]$.
We also know that $\sin \left(\frac{\pi}{6}\right) = \frac{1}{2}$.

Using the property $\sin^{-1}(-x) = -\sin^{-1} x$:
$y = -\sin^{-1} \left(\frac{1}{2}\right)$
$y = -\frac{\pi}{6}$

Since $-\frac{\pi}{6} \in [-\frac{\pi}{2}, \frac{\pi}{2}]$, it is the principal value.
Therefore, the principal value of $\sin^{-1} \left(-\frac{1}{2}\right)$ is $\boxed{-\frac{\pi}{6}}$. [9]

### **Question 11: Find the value of $\tan^{-1}(1) + \cos^{-1}\left(-\frac{1}{2}\right) + \sin^{-1}\left(-\frac{1}{2}\right)$**

**Thought Process:**
1.  This is a combination of three principal value problems. Solve each part individually first.
2.  **$\tan^{-1}(1)$:** Positive input, $\tan(\frac{\pi}{4}) = 1$. Range is $(-\frac{\pi}{2}, \frac{\pi}{2})$. So, $\frac{\pi}{4}$.
3.  **$\cos^{-1}\left(-\frac{1}{2}\right)$:** Negative input. Use $\pi - \cos^{-1}\left(\frac{1}{2}\right) = \pi - \frac{\pi}{3} = \frac{2\pi}{3}$.
4.  **$\sin^{-1}\left(-\frac{1}{2}\right)$:** Negative input. Use $-\sin^{-1}\left(\frac{1}{2}\right) = -\frac{\pi}{6}$.
5.  **Add the results:** Sum the three principal values.
6.  **Alternative (using a property):** Notice that $\cos^{-1}(-x) + \sin^{-1}(-x)$ looks like it might simplify. We know $\sin^{-1}x + \cos^{-1}x = \frac{\pi}{2}$. Can we use that?
    *   $\sin^{-1}\left(-\frac{1}{2}\right) + \cos^{-1}\left(-\frac{1}{2}\right) = -\sin^{-1}\left(\frac{1}{2}\right) + \pi - \cos^{-1}\left(\frac{1}{2}\right)$
    *   $= \pi - (\sin^{-1}\left(\frac{1}{2}\right) + \cos^{-1}\left(\frac{1}{2}\right))$
    *   $= \pi - \frac{\pi}{2} = \frac{\pi}{2}$.
    *   This is a much faster way!

**Solution:**
Let's evaluate each term:

1.  **For $\tan^{-1}(1)$:**
    Let $y_1 = \tan^{-1}(1)$. Then $\tan y_1 = 1$.
    The principal value branch for $\tan^{-1} x$ is $(-\frac{\pi}{2}, \frac{\pi}{2})$.
    Since $\tan\left(\frac{\pi}{4}\right) = 1$ and $\frac{\pi}{4} \in (-\frac{\pi}{2}, \frac{\pi}{2})$, $y_1 = \frac{\pi}{4}$.

2.  **For $\cos^{-1}\left(-\frac{1}{2}\right)$:**
    Let $y_2 = \cos^{-1}\left(-\frac{1}{2}\right)$. Then $\cos y_2 = -\frac{1}{2}$.
    The principal value branch for $\cos^{-1} x$ is $[0, \pi]$.
    Using the property $\cos^{-1}(-x) = \pi - \cos^{-1} x$:
    $y_2 = \pi - \cos^{-1}\left(\frac{1}{2}\right) = \pi - \frac{\pi}{3} = \frac{2\pi}{3}$.

3.  **For $\sin^{-1}\left(-\frac{1}{2}\right)$:**
    Let $y_3 = \sin^{-1}\left(-\frac{1}{2}\right)$. Then $\sin y_3 = -\frac{1}{2}$.
    The principal value branch for $\sin^{-1} x$ is $[-\frac{\pi}{2}, \frac{\pi}{2}]$.
    Using the property $\sin^{-1}(-x) = -\sin^{-1} x$:
    $y_3 = -\sin^{-1}\left(\frac{1}{2}\right) = -\frac{\pi}{6}$.

Now, add the values:
$\tan^{-1}(1) + \cos^{-1}\left(-\frac{1}{2}\right) + \sin^{-1}\left(-\frac{1}{2}\right) = \frac{\pi}{4} + \frac{2\pi}{3} + \left(-\frac{\pi}{6}\right)$

Find a common denominator, which is 12:
$= \frac{3\pi}{12} + \frac{8\pi}{12} - \frac{2\pi}{12}$
$= \frac{3\pi + 8\pi - 2\pi}{12} = \frac{9\pi}{12} = \frac{3\pi}{4}$

**Alternatively (using the complementary property):**
We know $\sin^{-1} x + \cos^{-1} x = \frac{\pi}{2}$.
The expression is $\tan^{-1}(1) + \left(\cos^{-1}\left(-\frac{1}{2}\right) + \sin^{-1}\left(-\frac{1}{2}\right)\right)$.
Let $x = -\frac{1}{2}$. Since $x \in [-1, 1]$, we can use the property for the terms in the parenthesis.
$\cos^{-1}\left(-\frac{1}{2}\right) + \sin^{-1}\left(-\frac{1}{2}\right) = \frac{\pi}{2}$.

So, the expression becomes:
$\tan^{-1}(1) + \frac{\pi}{2}$
We know $\tan^{-1}(1) = \frac{\pi}{4}$.
$= \frac{\pi}{4} + \frac{\pi}{2} = \frac{\pi}{4} + \frac{2\pi}{4} = \frac{3\pi}{4}$.

Therefore, the value of $\tan^{-1}(1) + \cos^{-1}\left(-\frac{1}{2}\right) + \sin^{-1}\left(-\frac{1}{2}\right)$ is $\boxed{\frac{3\pi}{4}}$. [9]

### **Question 12: Find the value of $\cos^{-1}\left(\frac{1}{2}\right) + 2\sin^{-1}\left(\frac{1}{2}\right)$**

**Thought Process:**
1.  Evaluate each inverse function separately. Both inputs are positive, so no negative argument properties needed.
2.  **$\cos^{-1}\left(\frac{1}{2}\right)$:** What angle has a cosine of $\frac{1}{2}$? It's $\frac{\pi}{3}$.
3.  **$\sin^{-1}\left(\frac{1}{2}\right)$:** What angle has a sine of $\frac{1}{2}$? It's $\frac{\pi}{6}$.
4.  **Substitute and calculate:** Plug these values into the expression.
5.  **Alternative (using a property):** We can split $2\sin^{-1}\left(\frac{1}{2}\right)$ into $\sin^{-1}\left(\frac{1}{2}\right) + \sin^{-1}\left(\frac{1}{2}\right)$. Then group with $\cos^{-1}\left(\frac{1}{2}\right)$ to use the complementary property $\sin^{-1}x + \cos^{-1}x = \frac{\pi}{2}$.

**Solution:**
Let's evaluate each term:

1.  **For $\cos^{-1}\left(\frac{1}{2}\right)$:**
    Let $y_1 = \cos^{-1}\left(\frac{1}{2}\right)$. Then $\cos y_1 = \frac{1}{2}$.
    The principal value branch for $\cos^{-1} x$ is $[0, \pi]$.
    Since $\cos\left(\frac{\pi}{3}\right) = \frac{1}{2}$ and $\frac{\pi}{3} \in [0, \pi]$, $y_1 = \frac{\pi}{3}$.

2.  **For $\sin^{-1}\left(\frac{1}{2}\right)$:**
    Let $y_2 = \sin^{-1}\left(\frac{1}{2}\right)$. Then $\sin y_2 = \frac{1}{2}$.
    The principal value branch for $\sin^{-1} x$ is $[-\frac{\pi}{2}, \frac{\pi}{2}]$.
    Since $\sin\left(\frac{\pi}{6}\right) = \frac{1}{2}$ and $\frac{\pi}{6} \in [-\frac{\pi}{2}, \frac{\pi}{2}]$, $y_2 = \frac{\pi}{6}$.

Now, substitute these values into the expression:
$\cos^{-1}\left(\frac{1}{2}\right) + 2\sin^{-1}\left(\frac{1}{2}\right) = \frac{\pi}{3} + 2\left(\frac{\pi}{6}\right)$
$= \frac{\pi}{3} + \frac{2\pi}{6}$
$= \frac{\pi}{3} + \frac{\pi}{3}$
$= \frac{2\pi}{3}$

**Alternatively (using the complementary property):**
The expression is $\cos^{-1}\left(\frac{1}{2}\right) + \sin^{-1}\left(\frac{1}{2}\right) + \sin^{-1}\left(\frac{1}{2}\right)$.
We know that $\cos^{-1} x + \sin^{-1} x = \frac{\pi}{2}$.
Here $x = \frac{1}{2}$, which is in $[-1, 1]$.
So, $\left(\cos^{-1}\left(\frac{1}{2}\right) + \sin^{-1}\left(\frac{1}{2}\right)\right) + \sin^{-1}\left(\frac{1}{2}\right) = \frac{\pi}{2} + \sin^{-1}\left(\frac{1}{2}\right)$
$= \frac{\pi}{2} + \frac{\pi}{6}$
$= \frac{3\pi}{6} + \frac{\pi}{6} = \frac{4\pi}{6} = \frac{2\pi}{3}$.

Therefore, the value of $\cos^{-1}\left(\frac{1}{2}\right) + 2\sin^{-1}\left(\frac{1}{2}\right)$ is $\boxed{\frac{2\pi}{3}}$. [9]

### **Question 14: $\tan^{-1} \sqrt{3} - \sec^{-1} (-2)$ is equal to**
**(A) $\pi$**
**(B) $-\frac{\pi}{3}$**
**(C) $\frac{\pi}{3}$**
**(D) $\frac{2\pi}{3}$**

**Thought Process:**
1.  Evaluate each inverse function separately.
2.  **$\tan^{-1} \sqrt{3}$:** Positive input. What angle has a tangent of $\sqrt{3}$? It's $\frac{\pi}{3}$. Range $(-\frac{\pi}{2}, \frac{\pi}{2})$.
3.  **$\sec^{-1} (-2)$:** Negative input. For $\sec^{-1}$, the property is $\sec^{-1}(-x) = \pi - \sec^{-1} x$.
    *   First, find $\sec^{-1}(2)$. Convert to cosine: $\cos^{-1}(\frac{1}{2})$, which is $\frac{\pi}{3}$.
    *   So, $\sec^{-1}(-2) = \pi - \frac{\pi}{3} = \frac{2\pi}{3}$. Range $[0, \pi] - \{\frac{\pi}{2}\}$.
4.  **Subtract the results:** $\frac{\pi}{3} - \frac{2\pi}{3}$.

**Solution:**
Let's evaluate each term:

1.  **For $\tan^{-1} \sqrt{3}$:**
    Let $y_1 = \tan^{-1} \sqrt{3}$. Then $\tan y_1 = \sqrt{3}$.
    The principal value branch for $\tan^{-1} x$ is $(-\frac{\pi}{2}, \frac{\pi}{2})$.
    Since $\tan\left(\frac{\pi}{3}\right) = \sqrt{3}$ and $\frac{\pi}{3} \in (-\frac{\pi}{2}, \frac{\pi}{2})$, $y_1 = \frac{\pi}{3}$.

2.  **For $\sec^{-1} (-2)$:**
    Let $y_2 = \sec^{-1} (-2)$. Then $\sec y_2 = -2$.
    The principal value branch for $\sec^{-1} x$ is $[0, \pi] - \{\frac{\pi}{2}\}$.
    Using the property $\sec^{-1}(-x) = \pi - \sec^{-1} x$:
    $y_2 = \pi - \sec^{-1} (2)$.
    To find $\sec^{-1}(2)$, we consider $\cos^{-1}\left(\frac{1}{2}\right)$, which is $\frac{\pi}{3}$.
    So, $y_2 = \pi - \frac{\pi}{3} = \frac{2\pi}{3}$.

Now, calculate the expression:
$\tan^{-1} \sqrt{3} - \sec^{-1} (-2) = \frac{\pi}{3} - \frac{2\pi}{3}$
$= -\frac{\pi}{3}$

Therefore, the correct option is **(B)**. [9]

2.  **Suggest a substitution:** This suggests letting $x = \sin\theta$.
3.  **Determine $\theta$'s range:** If $x = \sin\theta$, then $\theta = \sin^{-1}x$. Given $x \in [-\frac{1}{2}, \frac{1}{2}]$, what is the range for $\theta$? 4.  **Work with RHS:** Substitute $x = \sin\theta$ into the RHS and simplify using the $\sin 3\theta$ identity.

6.  **Substitute back:** Replace $\theta$ with $\sin^{-1}x$.

**Solution:**
Let $x = \sin \theta$.
Then $\theta = \sin^{-1} x$.

Given the domain $x \in [-\frac{1}{2}, \frac{1}{2}]$.
Since $\theta = \sin^{-1} x$, the range of $\sin^{-1} x$ is $[-\frac{\pi}{2}, \frac{\pi}{2}]$.
For $x = -\frac{1}{2}$, $\theta = -\frac{\pi}{6}$.
For $x = \frac{1}{2}$, $\theta = \frac{\pi}{6}$.
So, $\theta \in [-\frac{\pi}{6}, \frac{\pi}{6}]$.

Consider the Right Hand Side (RHS):
RHS = $\sin^{-1} (3x - 4x^3)$
Substitute $x = \sin \theta$:
RHS = $\sin^{-1} (3\sin \theta - 4\sin^3 \theta)$

Using the trigonometric identity $\sin 3\theta = 3\sin \theta - 4\sin^3 \theta$:
RHS = $\sin^{-1} (\sin 3\theta)$

$-\frac{\pi}{2} \le 3\theta \le \frac{\pi}{2}$

### **Question 3: Write $\tan^{-1} \left(\frac{\sqrt{1+x^2} - 1}{x}\right)$, $x \neq 0$ in the simplest form.**

**Thought Process:**
1.  **Recognize the structure with $\sqrt{1+x^2}$:** This type of expression often suggests a trigonometric substitution involving $\tan\theta$ or $\cot\theta$ because $1+\tan^2\theta = \sec^2\theta$ and $1+\cot^2\theta = \csc^2\theta$. The goal is to simplify the square root.
2.  **Suggest substitution:** Let $x = \tan\theta$. This means $\theta = \tan^{-1}x$.
3.  **Simplify the inner expression:** Substitute $x = \tan\theta$ into $\frac{\sqrt{1+x^2} - 1}{x}$ and simplify using trig identities.
4.  **Apply inverse identity property:** You should get $\tan^{-1}(\tan A)$ for some $A$. For this to be $A$, $A$ must be in the principal value branch of $\tan^{-1}$, i.e., $(-\frac{\pi}{2}, \frac{\pi}{2})$.

**Solution:**
Let $x = \tan \theta$. Then $\theta = \tan^{-1} x$.
Since $x \neq 0$, $\theta \neq 0$.
Substitute $x = \tan \theta$ into the expression:
$$ \tan^{-1} \left(\frac{\sqrt{1+\tan^2 \theta} - 1}{\tan \theta}\right) $$
We know that $1+\tan^2 \theta = \sec^2 \theta$.
$$ = \tan^{-1} \left(\frac{\sqrt{\sec^2 \theta} - 1}{\tan \theta}\right) $$
Since $\tan^{-1}x$ has range $(-\pi/2, \pi/2)$, we can assume $\theta$ is in this range. In this range, $\sec\theta$ is positive. So $\sqrt{\sec^2\theta} = \sec\theta$.
$$ = \tan^{-1} \left(\frac{\sec \theta - 1}{\tan \theta}\right) $$
Now, express $\sec\theta$ and $\tan\theta$ in terms of $\sin\theta$ and $\cos\theta$:
$$ = \tan^{-1} \left(\frac{\frac{1}{\cos \theta} - 1}{\frac{\sin \theta}{\cos \theta}}\right) $$
$$ = \tan^{-1} \left(\frac{\frac{1-\cos \theta}{\cos \theta}}{\frac{\sin \theta}{\cos \theta}}\right) $$
Cancel out $\cos\theta$ from numerator and denominator:
$$ = \tan^{-1} \left(\frac{1-\cos \theta}{\sin \theta}\right) $$
Now, use the half-angle identities:
$1-\cos \theta = 2\sin^2 \left(\frac{\theta}{2}\right)$
$\sin \theta = 2\sin \left(\frac{\theta}{2}\right) \cos \left(\frac{\theta}{2}\right)$
$$ = \tan^{-1} \left(\frac{2\sin^2 \left(\frac{\theta}{2}\right)}{2\sin \left(\frac{\theta}{2}\right) \cos \left(\frac{\theta}{2}\right)}\right) $$
$$ = \tan^{-1} \left(\frac{\sin \left(\frac{\theta}{2}\right)}{\cos \left(\frac{\theta}{2}\right)}\right) $$
$$ = \tan^{-1} \left(\tan \left(\frac{\theta}{2}\right)\right) $$
Since $x \neq 0$, $\theta \neq 0$. For $\theta \in (-\frac{\pi}{2}, \frac{\pi}{2})$, we have $\frac{\theta}{2} \in (-\frac{\pi}{4}, \frac{\pi}{4})$. This range is within $(-\frac{\pi}{2}, \frac{\pi}{2})$, the principal value branch for $\tan^{-1}x$.
Therefore, $\tan^{-1} \left(\tan \left(\frac{\theta}{2}\right)\right) = \frac{\theta}{2}$.
Substitute back $\theta = \tan^{-1} x$:
$$ = \frac{1}{2} \tan^{-1} x $$

The simplest form is $\boxed{\frac{1}{2} \tan^{-1} x}$. [12]

### **Question 5: Write $\tan^{-1} \left(\frac{\cos x - \sin x}{\cos x + \sin x}\right)$, $-\frac{\pi}{4} < x < \frac{3\pi}{4}$ in the simplest form.**

**Thought Process:**

$$ = \tan^{-1} \left(\frac{1 - \tan x}{1 + \tan x}\right) $$

1.  **Recognize the forms:** $\frac{2x}{1+x^2}$ and $\frac{1-y^2}{1+y^2}$ are classic structures for $2\tan^{-1}x$ properties.
    *   $\sin^{-1}\left(\frac{2x}{1+x^2}\right)$ reminds us of $2\tan^{-1}x = \sin^{-1}\left(\frac{2x}{1+x^2}\right)$.
    *   $\cos^{-1}\left(\frac{1-y^2}{1+y^2}\right)$ reminds us of $2\tan^{-1}y = \cos^{-1}\left(\frac{1-y^2}{1+y^2}\right)$.
2.  **Apply the properties:** Substitute these directly.
3.  **Check conditions:**
    *   For $2\tan^{-1}x = \sin^{-1}\left(\frac{2x}{1+x^2}\right)$, the condition is $|x|\le 1$. The problem states $|x|<1$, so it's valid.
    *   For $2\tan^{-1}y = \cos^{-1}\left(\frac{1-y^2}{1+y^2}\right)$, the condition is $y \ge 0$. The problem states $y>0$, so it's valid.
4.  **Simplify:** Add the terms and multiply by $\frac{1}{2}$.
5.  **Further simplification (if possible):** If we end up with $2(\tan^{-1}x + \tan^{-1}y)$, we might use the sum formula for $\tan^{-1}$.

**Solution:**
We are given the expression $\frac{1}{2} \left[\sin^{-1} \left(\frac{2x}{1+x^2}\right) + \cos^{-1} \left(\frac{1-y^2}{1+y^2}\right)\right]$ with conditions $|x|<1, y>0$ and $xy<1$.

From our properties of inverse trigonometric functions (Property Set 5):
1.  We know $2\tan^{-1} x = \sin^{-1} \left(\frac{2x}{1+x^2}\right)$, if $|x| \le 1$.
    The given condition $|x|<1$ satisfies this.
    So, $\sin^{-1} \left(\frac{2x}{1+x^2}\right) = 2\tan^{-1} x$.

2.  We know $2\tan^{-1} y = \cos^{-1} \left(\frac{1-y^2}{1+y^2}\right)$, if $y \ge 0$.
    The given condition $y>0$ satisfies this.
    So, $\cos^{-1} \left(\frac{1-y^2}{1+y^2}\right) = 2\tan^{-1} y$.

Substitute these back into the given expression:
$$ \frac{1}{2} \left[2\tan^{-1} x + 2\tan^{-1} y\right] $$
Factor out 2:
$$ \frac{1}{2} \times 2 \left[\tan^{-1} x + \tan^{-1} y\right] $$
$$ = \tan^{-1} x + \tan^{-1} y $$
Now, use the sum formula for inverse tangents (Property Set 5):
$\tan^{-1} x + \tan^{-1} y = \tan^{-1} \left(\frac{x+y}{1-xy}\right)$, if $xy < 1$.
The given condition $xy<1$ satisfies this.

So, the expression simplifies to $\boxed{\tan^{-1} \left(\frac{x+y}{1-xy}\right)}$. [12]

### **Question 10: Find the value of $\sin^{-1} \left(\sin \frac{2\pi}{3}\right)$**

**Thought Process:**
1.  **Identify the property:** This is of the form $\sin^{-1}(\sin x)$. We know $\sin^{-1}(\sin x) = x$ *if* $x$ is in the principal value branch of $\sin^{-1}x$, which is $[-\frac{\pi}{2}, \frac{\pi}{2}]$.
2.  **Check the condition:** Is $\frac{2\pi}{3}$ in $[-\frac{\pi}{2}, \frac{\pi}{2}]$? No, $\frac{2\pi}{3} = 120^\circ$, which is outside the range $[-90^\circ, 90^\circ]$.
3.  **Adjust the angle:** Since it's outside the range, we need to find an angle *within* the range that has the same sine value.
    *   $\frac{2\pi}{3}$ is in Quadrant II. Sine is positive in QII.
    *   The reference angle for $\frac{2\pi}{3}$ is $\pi - \frac{2\pi}{3} = \frac{\pi}{3}$.
    *   So, $\sin\left(\frac{2\pi}{3}\right) = \sin\left(\frac{\pi}{3}\right)$.
4.  **Apply the property with the adjusted angle:** Now, $\sin^{-1}\left(\sin \frac{\pi}{3}\right)$. Is $\frac{\pi}{3}$ in $[-\frac{\pi}{2}, \frac{\pi}{2}]$? Yes.

**Solution:**
We are asked to find the value of $\sin^{-1} \left(\sin \frac{2\pi}{3}\right)$.
We know that $\sin^{-1}(\sin x) = x$ only if $x \in [-\frac{\pi}{2}, \frac{\pi}{2}]$.
Here, $x = \frac{2\pi}{3}$. Since $\frac{2\pi}{3}$ ($120^\circ$) is not in $[-\frac{\pi}{2}, \frac{\pi}{2}]$ (i.e., $[-90^\circ, 90^\circ]$), we cannot directly apply the property.

We need to find an angle in $[-\frac{\pi}{2}, \frac{\pi}{2}]$ that has the same sine value as $\frac{2\pi}{3}$.
We know that $\sin(\pi - \theta) = \sin\theta$.
So, $\sin\left(\frac{2\pi}{3}\right) = \sin\left(\pi - \frac{2\pi}{3}\right) = \sin\left(\frac{\pi}{3}\right)$.
Now, the expression becomes $\sin^{-1}\left(\sin \frac{\pi}{3}\right)$.
Since $\frac{\pi}{3} \in [-\frac{\pi}{2}, \frac{\pi}{2}]$ (i.e., $60^\circ \in [-90^\circ, 90^\circ]$), we can apply the property.

Therefore, $\sin^{-1} \left(\sin \frac{2\pi}{3}\right) = \frac{\pi}{3}$.

The value is $\boxed{\frac{\pi}{3}}$. [13]

### **Question 13: Find the value of $\tan \left(\frac{1}{2} \left(\sin^{-1} \frac{2x}{1+x^2} + \cos^{-1} \frac{1-y^2}{1+y^2}\right)\right)$, where $|x|<1, y>0$ and $xy<1$.**

**Thought Process:**
1.  This is a slightly modified version of Question 9.
2.  **Evaluate the inner expression first:** We already simplified $\frac{1}{2} \left[\sin^{-1} \left(\frac{2x}{1+x^2}\right) + \cos^{-1} \left(\frac{1-y^2}{1+y^2}\right)\right]$ in Question 9 to $\tan^{-1} x + \tan^{-1} y$, which further simplifies to $\tan^{-1} \left(\frac{x+y}{1-xy}\right)$.
3.  **Substitute into the outer function:** Then apply the outer $\tan$ function to this simplified inverse tangent.
4.  **Apply identity:** $\tan(\tan^{-1} A) = A$, *if* $A$ is in the domain of $\tan^{-1}$ (which is $\mathbb{R}$).

**Solution:**
We need to find the value of $\tan \left(\frac{1}{2} \left(\sin^{-1} \frac{2x}{1+x^2} + \cos^{-1} \frac{1-y^2}{1+y^2}\right)\right)$.
We already simplified the inner argument from Question 9:
Let $A = \frac{1}{2} \left(\sin^{-1} \frac{2x}{1+x^2} + \cos^{-1} \frac{1-y^2}{1+y^2}\right)$.
Using the properties $2\tan^{-1} x = \sin^{-1} \left(\frac{2x}{1+x^2}\right)$ (for $|x|<1$) and $2\tan^{-1} y = \cos^{-1} \left(\frac{1-y^2}{1+y^2}\right)$ (for $y>0$):
$$ A = \frac{1}{2} \left(2\tan^{-1} x + 2\tan^{-1} y\right) $$
$$ A = \frac{1}{2} \times 2 (\tan^{-1} x + \tan^{-1} y) $$
$$ A = \tan^{-1} x + \tan^{-1} y $$
Using the sum property $\tan^{-1} x + \tan^{-1} y = \tan^{-1} \left(\frac{x+y}{1-xy}\right)$ (for $xy<1$):
$$ A = \tan^{-1} \left(\frac{x+y}{1-xy}\right) $$
Now, substitute this back into the original expression:
$$ \tan(A) = \tan\left(\tan^{-1} \left(\frac{x+y}{1-xy}\right)\right) $$
Using the property $\tan(\tan^{-1} A) = A$ (since $\frac{x+y}{1-xy}$ is a real number):
$$ = \frac{x+y}{1-xy} $$

The value is $\boxed{\frac{x+y}{1-xy}}$. [12]

### **Question 14: Find the value of $\sin^{-1}(1-x) - 2\sin^{-1} x = \frac{\pi}{2}$, then $x$ is equal to**
**(A) $0, \frac{1}{2}$**
**(B) $1, \frac{1}{2}$**
**(C) $0$**
**(D) $\frac{1}{2}$**

**Thought Process:**
1.  **Rearrange the equation:** Isolate one of the $\sin^{-1}$ terms, preferably the one with $2\sin^{-1}x$.
    $2\sin^{-1} x = \sin^{-1}(1-x) - \frac{\pi}{2}$.
2.  **Apply sine to both sides:** To get rid of the inverse sine functions, apply $\sin(\dots)$ to both sides.
    $\sin(2\sin^{-1} x) = \sin(\sin^{-1}(1-x) - \frac{\pi}{2})$.
3.  **Simplify LHS:** Use the $2\tan^{-1}x = \sin^{-1}(\frac{2x}{1+x^2})$ form, or let $\sin^{-1}x = \theta$ and use $\sin(2\theta) = 2\sin\theta\cos\theta$.
    Let $\sin^{-1} x = \theta$. Then $x = \sin\theta$. Also, $\cos\theta = \sqrt{1-\sin^2\theta} = \sqrt{1-x^2}$ (assuming $\theta \in [-\pi/2, \pi/2]$).
    So, LHS becomes $\sin(2\theta) = 2\sin\theta\cos\theta = 2x\sqrt{1-x^2}$.
4.  **Simplify RHS:** Use $\sin(A-B) = \sin A \cos B - \cos A \sin B$. Here $A = \sin^{-1}(1-x)$ and $B = \frac{\pi}{2}$.
    $\sin(\sin^{-1}(1-x) - \frac{\pi}{2}) = \sin(\sin^{-1}(1-x))\cos(\frac{\pi}{2}) - \cos(\sin^{-1}(1-x))\sin(\frac{\pi}{2})$.
    We know $\cos(\frac{\pi}{2}) = 0$ and $\sin(\frac{\pi}{2}) = 1$.
    Also, $\sin(\sin^{-1}(1-x)) = 1-x$.
    And $\cos(\sin^{-1}(1-x)) = \sqrt{1-(1-x)^2}$.
    So, RHS becomes $(1-x)(0) - \sqrt{1-(1-x)^2}(1) = -\sqrt{1-(1-x)^2}$.
5.  **Form the equation:** $2x\sqrt{1-x^2} = -\sqrt{1-(1-x)^2}$.
6.  **Solve the equation:** Square both sides (watch for extraneous solutions!). Simplify and solve for $x$.
    *   **Crucial step: Check domain conditions for $\sin^{-1} x$ and $\sin^{-1}(1-x)$.**
        *   For $\sin^{-1} x$, we need $x \in [-1, 1]$.
        *   For $\sin^{-1}(1-x)$, we need $1-x \in [-1, 1]$, which means $-1 \le 1-x \le 1$.
            *   $-1 \le 1-x \implies x \le 2$.
            *   $1-x \le 1 \implies x \ge 0$.
            *   So, we need $x \in [0, 1]$ for both to be valid.

**Solution:**
We are given the equation $\sin^{-1}(1-x) - 2\sin^{-1} x = \frac{\pi}{2}$.

Rearrange the equation:
$\sin^{-1}(1-x) = \frac{\pi}{2} + 2\sin^{-1} x$.

Let $A = \sin^{-1}(1-x)$ and $B = 2\sin^{-1} x$.
Taking sine on both sides:
$\sin(\sin^{-1}(1-x)) = \sin\left(\frac{\pi}{2} + 2\sin^{-1} x\right)$.

LHS: $\sin(\sin^{-1}(1-x)) = 1-x$. (Valid for $1-x \in [-1,1]$)

RHS: Using $\sin(\frac{\pi}{2} + \theta) = \cos\theta$:
$\sin\left(\frac{\pi}{2} + 2\sin^{-1} x\right) = \cos(2\sin^{-1} x)$.

Let $\theta = \sin^{-1} x$. Then $x = \sin\theta$. And $\cos\theta = \sqrt{1-x^2}$ (assuming principal values, so $\theta \in [-\pi/2, \pi/2]$ and $\cos\theta \ge 0$).
So, RHS becomes $\cos(2\theta)$.
We know $\cos(2\theta) = 1 - 2\sin^2\theta$.
Substitute $\sin\theta = x$:
RHS = $1 - 2x^2$.

Now, equate LHS and RHS:
$1 - x = 1 - 2x^2$
$2x^2 - x = 0$
$x(2x - 1) = 0$
This gives two possible solutions: $x = 0$ or $x = \frac{1}{2}$.

**Check for validity:**
We need to ensure these solutions are valid within the domains of the inverse functions.
1.  For $\sin^{-1} x$: $x \in [-1, 1]$. Both $0$ and $\frac{1}{2}$ are in this range.
2.  For $\sin^{-1} (1-x)$: $1-x \in [-1, 1]$, which means $0 \le x \le 2$. Both $0$ and $\frac{1}{2}$ are in this range.

Additionally, when we made the substitution $\sin^{-1}(1-x) = \frac{\pi}{2} + 2\sin^{-1} x$, the LHS, $\sin^{-1}(1-x)$, has a range of $[-\frac{\pi}{2}, \frac{\pi}{2}]$.
The RHS, $\frac{\pi}{2} + 2\sin^{-1} x$.
Since $\sin^{-1} x \in [-\frac{\pi}{2}, \frac{\pi}{2}]$,
$2\sin^{-1} x \in [-\pi, \pi]$.
So, $\frac{\pi}{2} + 2\sin^{-1} x \in [-\frac{\pi}{2}, \frac{3\pi}{2}]$.

We need $\sin^{-1}(1-x) = \frac{\pi}{2} + 2\sin^{-1} x$.
For $x=0$:
LHS: $\sin^{-1}(1-0) = \sin^{-1}(1) = \frac{\pi}{2}$.
RHS: $\frac{\pi}{2} + 2\sin^{-1}(0) = \frac{\pi}{2} + 2(0) = \frac{\pi}{2}$.
LHS = RHS. So, $x=0$ is a valid solution.

For $x=\frac{1}{2}$:
LHS: $\sin^{-1}(1-\frac{1}{2}) = \sin^{-1}(\frac{1}{2}) = \frac{\pi}{6}$.
RHS: $\frac{\pi}{2} + 2\sin^{-1}(\frac{1}{2}) = \frac{\pi}{2} + 2\left(\frac{\pi}{6}\right) = \frac{\pi}{2} + \frac{\pi}{3} = \frac{3\pi + 2\pi}{6} = \frac{5\pi}{6}$.
LHS ($\frac{\pi}{6}$) $\neq$ RHS ($\frac{5\pi}{6}$). So, $x=\frac{1}{2}$ is an extraneous solution.

The problem asks "then $x$ is equal to". This implies a single answer among the options.

Let's double check the step $\cos(2\sin^{-1}x)$. If $\sin^{-1}x = \theta$, then $\theta \in [-\pi/2, \pi/2]$. This means $2\theta \in [-\pi, \pi]$. In this interval, $\cos(2\theta) = 1-2\sin^2\theta$ is always valid.

The key check is for $\sin(A-B) = \sin A \cos B - \cos A \sin B$ and $\sin(A+B)$.
The equation $\sin^{-1}(1-x) - 2\sin^{-1} x = \frac{\pi}{2}$ means that the sum of these two angles is $\pi/2$.

Let's go back to the original equation $\sin^{-1}(1-x) = \frac{\pi}{2} + 2\sin^{-1} x$.
Let $\sin^{-1} x = \alpha$. So $x = \sin\alpha$.
The equation is $\sin^{-1}(1-x) = \frac{\pi}{2} + 2\alpha$.
For $\sin^{-1}(1-x)$ to be defined, $1-x \in [-1, 1] \implies x \in [0, 2]$.
For $\alpha$ to be defined, $x \in [-1, 1]$.
Combined domain: $x \in [0, 1]$.
The range of $\sin^{-1}(1-x)$ is $[-\pi/2, \pi/2]$.
So, we must have $\frac{\pi}{2} + 2\alpha \in [-\pi/2, \pi/2]$.
$\implies -\pi/2 \le \pi/2 + 2\alpha \le \pi/2$
$\implies -\pi \le 2\alpha \le 0$
$\implies -\pi/2 \le \alpha \le 0$.
Since $\alpha = \sin^{-1}x$, this means $x = \sin\alpha \in [-1, 0]$.

However, we already derived the combined domain for $x$ as $[0, 1]$.
The stricter condition $x \in [-1, 0]$ combined with $x \in [0, 1]$ implies $x=0$.
This means that if a solution exists, it must be $x=0$.

Let's re-verify the substitution steps.
$1-x = \sin(\frac{\pi}{2} + 2\alpha) = \cos(2\alpha) = 1-2\sin^2\alpha = 1-2x^2$.
This algebraic simplification is correct and yields $x=0$ or $x=1/2$.

Now, let's verify these solutions in the *original* equation:
**Case 1: $x = 0$**
LHS = $\sin^{-1}(1-0) - 2\sin^{-1}(0) = \sin^{-1}(1) - 2(0) = \frac{\pi}{2} - 0 = \frac{\pi}{2}$.
RHS = $\frac{\pi}{2}$.
LHS = RHS. So, $x=0$ is a solution.

**Case 2: $x = \frac{1}{2}$**
LHS = $\sin^{-1}(1-\frac{1}{2}) - 2\sin^{-1}(\frac{1}{2}) = \sin^{-1}(\frac{1}{2}) - 2\sin^{-1}(\frac{1}{2})$.
LHS = $-\sin^{-1}(\frac{1}{2}) = -\frac{\pi}{6}$.
RHS = $\frac{\pi}{2}$.
LHS ($\neq \frac{\pi}{2}$). So, $x=\frac{1}{2}$ is NOT a solution.

This type of "extraneous solution" happens when you square both sides of an equation (which we didn't explicitly do here, but deriving $\cos(2\sin^{-1}x)$ from $\sin(\frac{\pi}{2}+2\sin^{-1}x)$ carries similar potential for losing information about sign). More accurately, the step $\sin(A+B) = \cos A \sin B - \cos B \sin A$ when $B=\pi/2$ and $A$ being an inverse function means you have to be careful with the range of $A$.

However, the simpler and safer check is always to plug the potential solutions back into the original equation.

Therefore, the only valid solution is $x=0$.

The correct option is **(C)**. [14]
