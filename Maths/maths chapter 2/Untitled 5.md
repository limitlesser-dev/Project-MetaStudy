### **THE ESSENTIAL TRIGONOMETRY MEMORIZATION LIST**

**I. Basic Definitions (Right-Angled Triangle)**

For a right-angled triangle with angle $\theta$:
*   $\sin\theta = \frac{\text{Opposite}}{\text{Hypotenuse}}$
*   $\cos\theta = \frac{\text{Adjacent}}{\text{Hypotenuse}}$
*   $\tan\theta = \frac{\text{Opposite}}{\text{Adjacent}}$

**II. Reciprocal Identities**

*   $\csc\theta = \frac{1}{\sin\theta}$
*   $\sec\theta = \frac{1}{\cos\theta}$
*   $\cot\theta = \frac{1}{\tan\theta}$

**III. Quotient Identities**

*   $\tan\theta = \frac{\sin\theta}{\cos\theta}$
*   $\cot\theta = \frac{\cos\theta}{\sin\theta}$

**IV. Pythagorean Identities**

*   $\sin^2\theta + \cos^2\theta = 1$
*   $1 + \tan^2\theta = \sec^2\theta$
*   $1 + \cot^2\theta = \csc^2\theta$

**V. Domain and Range of Primary Trigonometric Functions**

| Function | Domain                                 | Range          |
| :------- | :------------------------------------- | :------------- |
| $\sin x$ | $\mathbb{R}$                           | $[-1, 1]$      |
| $\cos x$ | $\mathbb{R}$                           | $[-1, 1]$      |
| $\tan x$ | $\mathbb{R} - \{(2n+1)\frac{\pi}{2}\}$ | $\mathbb{R}$   |
| $\csc x$ | $\mathbb{R} - \{n\pi\}$                | $(-\infty, -1] \cup [1, \infty)$ |
| $\sec x$ | $\mathbb{R} - \{(2n+1)\frac{\pi}{2}\}$ | $(-\infty, -1] \cup [1, \infty)$ |
| $\cot x$ | $\mathbb{R} - \{n\pi\}$                | $\mathbb{R}$   |
*($n$ is an integer)*

**VI. Values for Common Angles (Degrees & Radians)**

| Angle ($\theta$) | $\sin\theta$      | $\cos\theta$      | $\tan\theta$      |
| :--------------- | :---------------- | :---------------- | :---------------- |
| $0^\circ \ (0)$  | $0$               | $1$               | $0$               |
| $30^\circ \ (\frac{\pi}{6})$ | $\frac{1}{2}$         | $\frac{\sqrt{3}}{2}$      | $\frac{1}{\sqrt{3}}$      |
| $45^\circ \ (\frac{\pi}{4})$ | $\frac{1}{\sqrt{2}}$      | $\frac{1}{\sqrt{2}}$      | $1$               |
| $60^\circ \ (\frac{\pi}{3})$ | $\frac{\sqrt{3}}{2}$      | $\frac{1}{2}$         | $\sqrt{3}$        |
| $90^\circ \ (\frac{\pi}{2})$ | $1$               | $0$               | Undefined         |
| $180^\circ \ (\pi)$ | $0$               | $-1$              | $0$               |
| $270^\circ \ (\frac{3\pi}{2})$ | $-1$              | $0$               | Undefined         |
| $360^\circ \ (2\pi)$ | $0$               | $1$               | $0$               |

**VII. Sign Rules (CAST Rule)**

*   **C** = Cosine is positive (Quadrant IV)
*   **A** = All are positive (Quadrant I)
*   **S** = Sine is positive (Quadrant II)
*   **T** = Tangent is positive (Quadrant III)

**VIII. Negative Angle Identities**

*   $\sin(-\theta) = -\sin\theta$
*   $\cos(-\theta) = \cos\theta$
*   $\tan(-\theta) = -\tan\theta$
*   $\csc(-\theta) = -\csc\theta$
*   $\sec(-\theta) = \sec\theta$
*   $\cot(-\theta) = -\cot\theta$

**IX. Co-function (Complementary Angle) Identities**

*   $\sin(\frac{\pi}{2} - \theta) = \cos\theta$
*   $\cos(\frac{\pi}{2} - \theta) = \sin\theta$
*   $\tan(\frac{\pi}{2} - \theta) = \cot\theta$
*   $\csc(\frac{\pi}{2} - \theta) = \sec\theta$
*   $\sec(\frac{\pi}{2} - \theta) = \csc\theta$
*   $\cot(\frac{\pi}{2} - \theta) = \tan\theta$

**X. Angle Addition and Subtraction Formulas (Compound Angle Formulas)**

*   $\sin(A+B) = \sin A \cos B + \cos A \sin B$
*   $\sin(A-B) = \sin A \cos B - \cos A \sin B$
*   $\cos(A+B) = \cos A \cos B - \sin A \sin B$
*   $\cos(A-B) = \cos A \cos B + \sin A \sin B$
*   $\tan(A+B) = \frac{\tan A + \tan B}{1 - \tan A \tan B}$
*   $\tan(A-B) = \frac{\tan A - \tan B}{1 + \tan A \tan B}$

**XI. Double Angle Formulas**

*   $\sin(2A) = 2\sin A \cos A$
*   $\cos(2A) = \cos^2 A - \sin^2 A$
*   $\cos(2A) = 2\cos^2 A - 1$
*   $\cos(2A) = 1 - 2\sin^2 A$
*   $\tan(2A) = \frac{2\tan A}{1 - \tan^2 A}$
*   **Useful variations (Half-angle squared):**
    *   $1 - \cos(2A) = 2\sin^2 A$
    *   $1 + \cos(2A) = 2\cos^2 A$

**XII. Triple Angle Formulas**

*   $\sin(3A) = 3\sin A - 4\sin^3 A$
*   $\cos(3A) = 4\cos^3 A - 3\cos A$
*   $\tan(3A) = \frac{3\tan A - \tan^3 A}{1 - 3\tan^2 A}$

**XIII. Half Angle Formulas (in terms of $\cos A$)**

*   $\sin\left(\frac{A}{2}\right) = \pm\sqrt{\frac{1 - \cos A}{2}}$
*   $\cos\left(\frac{A}{2}\right) = \pm\sqrt{\frac{1 + \cos A}{2}}$
*   $\tan\left(\frac{A}{2}\right) = \pm\sqrt{\frac{1 - \cos A}{1 + \cos A}} = \frac{1 - \cos A}{\sin A} = \frac{\sin A}{1 + \cos A}$
    *   *The sign depends on the quadrant of $A/2$*

**XIV. Sum-to-Product Formulas**

*   $\sin C + \sin D = 2\sin\left(\frac{C+D}{2}\right)\cos\left(\frac{C-D}{2}\right)$
*   $\sin C - \sin D = 2\cos\left(\frac{C+D}{2}\right)\sin\left(\frac{C-D}{2}\right)$
*   $\cos C + \cos D = 2\cos\left(\frac{C+D}{2}\right)\cos\left(\frac{C-D}{2}\right)$
*   $\cos C - \cos D = -2\sin\left(\frac{C+D}{2}\right)\sin\left(\frac{C-D}{2}\right)$

**XV. Product-to-Sum Formulas**

*   $2\sin A \cos B = \sin(A+B) + \sin(A-B)$
*   $2\cos A \sin B = \sin(A+B) - \sin(A-B)$
*   $2\cos A \cos B = \cos(A+B) + \cos(A-B)$
*   $2\sin A \sin B = \cos(A-B) - \cos(A+B)$

## The Complete Trigonometry Formula Cheat Sheet

### I. Fundamental Definitions & Identities

These are the most basic rules that define the relationships between the functions.

| Identity Type | Formula |

### II. Sum and Difference Formulas (Addition/Subtraction)

Used to break down angles like $\sin(A \pm B)$.

| Formula Name | Formula |
| :--- | :--- |
| **Sine of Sum/Difference** | $\sin(A + B) = \sin A \cos B + \cos A \sin B$ |
| | $\sin(A - B) = \sin A \cos B - \cos A \sin B$ |
| **Cosine of Sum/Difference** | $\cos(A + B) = \cos A \cos B - \sin A \sin B$ |
| | $\cos(A - B) = \cos A \cos B + \sin A \sin B$ |
| **Tangent of Sum/Difference** | $\mathbf{\tan(A + B) = \frac{\tan A + \tan B}{1 - \tan A \tan B}}$ |
| | $\mathbf{\tan(A - B) = \frac{\tan A - \tan B}{1 + \tan A \tan B}}$ |

### III. Multiple Angle Formulas (Double and Triple)

| Formula Name | Formula | Inverse Function Application (Substitution) |

### IV. Half-Angle Formulas (Rearranged Double-Angle)

These are the rearranged forms of the Cosine Double Angle identity, making the $1 \pm \cos x$ simplification possible.

| Identity Form | Formula (Rearranged from $\cos(2A)$) |
| :--- | :--- |
| **$1 - \cos x$** | $\mathbf{1 - \cos x = 2\sin^2 \left(\frac{x}{2}\right)}$ |
| **$1 + \cos x$** | $\mathbf{1 + \cos x = 2\cos^2 \left(\frac{x}{2}\right)}$ |
